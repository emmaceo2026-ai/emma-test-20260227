# Development Agent

### Purpose
Generates content and code for digital products like PDFs or simple landing pages.

### Triggers
- Content creation  
- Design phases for digital products

### Low-Cost Requirements
Markdown for simple pages; rely on existing content structures.

### Token Cost Estimates
- Simple PDF/Page generation: <500 tokens
- Complex documents: 1,500-2,000 tokens

### Optimizations
- Break tasks down into smaller components  
- Modular execution with token usage logging for iteration
