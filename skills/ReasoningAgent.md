# Reasoning Agent

### Purpose
Assists in decision-making and step-by-step planning for product development. Ideal for brainstorming and assessing product viability.

### Triggers
- Initial brainstorming
- Product viability assessments

### Low-Cost Requirements
Utilize internal knowledge graphs and cached data from memory files for simulations. Use structured templates for decision matrices.

### Token Cost Estimates
- Basic tasks: <500 tokens
- Comprehensive tasks: 1,500-2,000 tokens

### Optimizations
- Summarize inputs to focus on key elements  
- Break plans into modular tasks  
- Seek approval for escalated tasks  
- Post-execution logging of token usage for iteration
